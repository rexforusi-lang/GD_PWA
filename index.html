限制說明

          
    
            
  • 若公司網路禁止 Google Drive / Google API / Google Docs 匯出連結，本 PWA 也會被擋；請由 IT 以核准方式放行必要網域。
    • 
            
  • 大型資料夾會受瀏覽器記憶體、Drive API 配額、檔案權限與 OAuth token 時效影響；V1_5 採逐包或連續建立、最多 100 檔 / 1GB 分包、下載後自動或手動釋放暫存，避免一次保留所有 ZIP Blob。
    • 
            
  • Google Workspace 原生檔案需匯出；非檔案型服務如 Forms 可能無法下載。
    • 
          

        

      


      

        

          
使用流程

          
    
            
  1. 若是單一檔案：貼上分享連結，按「分析 / 掃描」，再按「單檔直連」。
    2. 
            
  2. 若是資料夾或要批次 ZIP：在 Google Cloud Console 啟用 Google Drive API。
    3. 
            
  3. 建立 OAuth 2.0「Web application」Client ID，授權來源加入 GitHub Pages 網址。
    4. 
            
  4. 把 Client ID 貼到設定頁並儲存。
    5. 
            
  5. 回到下載頁，登入 Google，貼上共用連結後掃描，勾選要下載的檔案；可按「建立第 1 包」手動逐包處理，或設定連續包數後按「連續建立/下載」。
    6.

drive.google.com/drive/folders/<folderId>

docs.google.com/document/d/<fileId>/edit
docs.google.com/spreadsheets/d/<fileId>/edit
docs.google.com/presentation/d/<fileId>/edit

        


        

          
合規替代方案

          
若 Zscaler 擋下載，建議請 IT 建立正式例外規則、使用企業核准的 Google Workspace 存取方式、或改用公司核准的檔案交換平台。本工具不會也不能繞過資安代理；免 API 直連也只是開啟 Google 官方下載頁。
